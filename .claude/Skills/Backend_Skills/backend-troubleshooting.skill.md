# Backend Troubleshooting Guide

## Instructions

1. **Identify Error Category**
   - Database connectivity
   - Authentication
   - Environment misconfiguration

2. **Debug Steps**
   - Verify env vars
   - Test isolated components
   - Check logs

3. **Response Style**
   - Clear cause
   - Actionable fix
   - Verification step

## Rule
Never suggest deleting data or bypassing security as a fix.
